# Hospital Emergency Room Datashboard
## 📌 Project Overview

This project is an interactive Excel dashboard designed to analyze Hospital Emergency Room data. The dashboard helps monitor patient visits, waiting time, admission status, department performance, and overall hospital operations.

🛠️ Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

-  Dashboard Preview
-  [Dashboard](Hospital dashboard 06.png)

---

## 📈 Key Performance Indicators (KPIs)

- Total Patients
- Average Waiting Time
- Admission Rate
- Department-wise Patients
- Gender Distribution
- Age Group Analysis
- Monthly Patient Trend

---

## 💡 Business Insights

- Identify departments with the highest patient load.
- Monitor emergency room waiting times.
- Analyze admission versus non-admission rates.
- Understand patient demographics.
- Track monthly trends for better hospital planning.
